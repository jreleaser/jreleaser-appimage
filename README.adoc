Follow these instructions to run JReleaser as an AppImage:

* Download the latest AppImage from the release page

[source]
[subs="attributes"]
----
$ curl https://github.com/jreleaser/jreleaser-appimage/releases/download/v{project-version}/jreleaser-{project-version}-x86_64.AppImage --output jreleaser
----

* Make it executable

```
$ chmod +x jreleaser
```

* Run it

```
$ ./jreleaser --help
```
